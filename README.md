## COMP3133_101543366_Assignment1

This project is an Employee Management System backend that I developed using Node.js, 
Express, GraphQL, MongoDB Atlas, and Cloudinary. The system allows users to sign up 
and log in, and once authenticated, they can add employees, upload employee photos, 
view all employee records, search employees by ID, update employee information, 
delete employees, and filter employees by designation or department.
